# SQL Data Cleaning Project

## Overview

This project demonstrates SQL-based data cleaning using a raw business dataset. The focus is on preparing messy data for reliable analysis and reporting.

---

## Business Problem

The goal of this project is to turn raw data into practical business insights. The analysis focuses on identifying patterns, measuring performance, and recommending actions that can improve decision-making.

## Data Cleaning Plan

1. Import the raw dataset.
2. Inspect column names, data types, and missing values.
3. Remove or flag duplicate records.
4. Standardize date, text, and numeric columns.
5. Create calculated fields needed for analysis.
6. Export a cleaned dataset to `data/cleaned/`.
7. Document every cleaning decision in the notebook and README.

---

## Analysis Plan

1. Load the cleaned dataset.
2. Calculate core KPIs.
3. Explore trends over time.
4. Segment the data by relevant business categories.
5. Create visuals that explain the key findings.
6. Build a final dashboard or summary report.
7. Translate findings into business recommendations.
